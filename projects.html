Game of Life

This is a React version of John Horton Conway's Game of Life. The challenge of this project was in its optimization. The amount of rerenders of the game board that were required made it difficult to render many generations in short period of time. Caching with hashmaps along with using the Canvas API got me the performance I was after.

The project also includes functionality to convert RLE (run length encoded) files to pattern objects that can be easily used by the application. RLE files are used by Game of Life enthusiasts to store patterns on the game board. I included two of my favorites in the deployed version of the application. Time permitting, I'll add more patterns in the future.

Note: This application is meant to be used on a PC.
